# Guess the number
web application

## default values
default values are:
- highest number you can guess is 100
- max tries are 10

you can change them by going to settings and insert the specified number you want and press the button

## to play
game here: https://torakh.github.io/Number_Guesser/
navigate to game
press the generate number button
insert guess
press check
you will get either correct number, you need to guess higher or lower
it says at the bottom max number and tries left
